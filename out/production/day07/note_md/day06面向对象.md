### 二维数组  
   * 1.定义：本质上二维数组是一个元素为一维数组的数组  
   * 2.声明：
        * 数据类型[][] 数组名;
        * int[][] arr;
        * int[] arr;
   * 3.初始化：开辟内存空间，并赋值
        * 方式一：静态初始化:我们给定元素数据，数组的长度由系统决定。
            * int[][] arr = new int[][]{{1},{2,4},{3,5,6}};
            * int[][] arr = {{1},{2,4},{3,5,6}};      
        * 方式二：动态初始化：我们给定长度，初始值由系统给出（byte/sort/int/long-->0 float/double/-->0.0 char-->0 boolean-->false）
            * int[][] arr = new int[3][2];//表实创建一个三行两列的二维数组
            * int[][] arr = new int[3][];//创建一个三行的二维数组，每行的列数未定义
            * arr[0] = new int[]{1}; 
            * arr[1] = new int[]{2,3};
            * arr[2] = new int[]{4,5,6};

### 面向对象  
```
    1.面向对象的概念：
        比较面向过程而言，注重实现的步骤，程序员参与功能的每个步骤
        面向对象，注重是由谁来完成功能的实现，程序员是个指挥者
    
    2.类与对象的概念
        对象：具体的事物，一类抽象事物的某个具体实例。
        类：抽象的概念，一类事物的抽象描述，类是对象的模板。
    
    3.类的定义
        修饰符 class 类名 { 类的成员 }   
        类的成员：
            成员变量-->属性（字段）
            成员方法-->行为和功能特征 
        示例：
            public class Student{
            String name;
            int age;
            public void stydy(){
                System.out.println("学习");    
            }    
        }
            
    4.类和对象的使用
        通过类来创建对象的格式：
            类名 对象名 = new 类名();
            Student s = new Student();
        对象的使用：
            s.name = "Tom";
            s.age = 18;
            System.out.println(s.name);
            System.out.println(s.name);
            s.study();

    5.对象的内存分析
        见day06imags

    6.成员变量的分类
        局部变量：定义位置在方法、代码块、参数中，作用范围是类的局部范围，内存加载在栈当中，生命周期随着方法的调用而加载，
        随着方法的结束而消失，只能由final修饰符
        没有默认初始值，所以要先声明初始化再使用。

        成员变量：定义在类的成员位置，方法外，作用范围是整个类，可以使用 public static private final等，有默认初始值（同数组）
            类变量：有static修饰符修饰，可以使用类名直接调用，整个类只有一份类变量，即通过此类创建的对象共享一份变量，
            加载在方法区，声明周期跟类同步，随着类的加载而加载。
                static country  = "中国";
            实例变量：没有static修饰，每个对象独有一份实例变量，加载内存位置是堆，声明周期跟对象同步的。
``` 
